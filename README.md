# Sea battle game js 

> jquery 3.x

> IE >= 8 

Play [demo](http://sea-battle-js.archebasov.ru).

## install
```bash
    yarn

```

## dev
```bash   
    yarn dev   # webapck watch on changes
    yarn serve # serve 8081

```

## build
```bash
    yarn build [--size] # build and watch production bundle size 

```
